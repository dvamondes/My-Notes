
- Config profile
<br>
```
$ git config --global user.name "NOME COMPLETO"
$ git config --global user.email "EMAIL@EMAIL"
```

- Commit
<br>
```
$ git commit -m "Commit Notes" -a
$ git push origin master
```

- Checkout
<br>
```
$ git clone [URL] .
```

- Update
```
if get conflits $ git reset --hard origin/master
$ git fetch --all
$ git pull origin master
```
